# wordnote
単語帳のアプリケーションのリポジトリ

## 概要
単語をTextAreaで入力できるようにした単語帳

## 使用言語
HTML， PHP， JavaScript， （MySQL）

## 使用フレームワーク
Bootstrap

## 環境
動作確認をしたブラウザ：Google Chrome
　クライアント：MacBook Air (M1, 2020)
